# 📖 About This Project

This project is a Kanban Board App built to help users manage their tasks in a simple and organized way.

Tasks are divided into three columns:

📝 To Do  
⚙️ Doing  
✅ Done  

The app is designed to be easy to use, responsive, and reliable, with data saved even after refreshing the page.

---

# 🌟 Features

## 🌐 Deployment
The app is deployed using Netlify.  
It is live and accessible from anywhere, and all features work in the deployed version.

## 🔄 API Fetching
Tasks are fetched from an online API:  
https://jsl-kanban-api.vercel.app/

- Users see a loading message while data loads  
- If there is a problem, an error message will be displayed  

## 💾 Local Storage
Tasks are saved in the browser using localStorage.

This means:
- Tasks stay even after refreshing  
- No data is lost  

## ✏️ Edit and Delete
Users can:
- Edit task title, description, and status  
- Delete tasks when no longer needed  

A confirmation message appears before deleting, and changes update instantly on the board.

## 📂 Sidebar
The sidebar contains important controls.

Users can:
- Open and close it  
- Use it on both desktop and mobile screens  

## 📱 Mobile Menu
On smaller screens:
- The sidebar becomes a menu  
- It can be opened from the top  
- It can be closed easily for a clean view  

## 🌙 Theme Toggle
Users can switch between:
🌞 Light mode  
🌙 Dark mode  

- Works across all devices  
- Ensures good readability  

## ⭐ Priority Feature
Tasks can have priority levels:

🔴 High  
🟡 Medium  
🟢 Low  

Priority is:
- Visible on each task  
- Saved in localStorage  
- Automatically sorted (High → Medium → Low)
